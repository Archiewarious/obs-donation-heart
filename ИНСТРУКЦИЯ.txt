СЕРДЦЕ-ДОНАТ для OBS — инструкция
==================================

Неоновое сердце с твоей картинкой, наполняется "водой" по мере сбора на
DonationAlerts. Данные живые. Помощник запускается сам вместе с OBS и
выключается сам после его закрытия — в фоне 24/7 ничего не висит.


ЧАСТЬ 1. Подключить скрипт в OBS (делается ОДИН раз)
---------------------------------------------------
  1. Открой OBS.
  2. Верхнее меню -> Инструменты (Tools).
  3. -> Скрипты (Scripts).
  4. В открывшемся окне слева вкладка "Скрипты".
  5. Внизу нажми "+".
  6. Зайди в папку этого проекта, выбери файл  heart-helper.lua  -> Открыть.
  7. Закрой окно скриптов.
  Теперь при каждом запуске OBS помощник поднимается сам.
  Открывай OBS как обычно (иконка / Steam) — больше ничего не нажимать.


ЧАСТЬ 2. Добавить сердце в сцену (один раз)
-------------------------------------------
  1. В блоке "Источники" нажми "+".
  2. Выбери "Браузер" (Browser) -> назови "Сердце" -> ОК.
  3. В свойствах:
       URL:     http://localhost:8123/heart-obs.html?obs=1
       Ширина:  360
       Высота:  480
     (галочку "Локальный файл" НЕ ставить)
  4. ОК. Перетащи/масштабируй сердце в сцене куда надо.


ПРОВЕРКА
--------
  Должно показать сердце и сумму. Сейчас ~4% (собрано 538 из 12222) — вода
  почти не видна, это реальный прогресс, с донатами поднимется.
  Если пусто/белое: ПКМ по источнику -> Обновить кэш; либо переоткрой OBS.


СМЕНИТЬ СБОР (самое простое)
----------------------------
  Открой файл  ссылка-на-сбор.txt  -> замени ссылку на новую -> сохрани.
  Всё. Перезапускать ничего не нужно, обновится за ~12 сек.
  Надпись и сумма подтянутся из нового сбора автоматически.

НАСТРОЙКИ ВИДА (в ссылке источника OBS)
--------------------------------------
  &size=420   размер сердца
  &title=0    спрятать надпись (по умолчанию надпись показана)
  &level=50   тест-уровень 50% (без живых данных, просто посмотреть)
  Поменять: ПКМ по источнику -> Свойства -> измени URL -> ОК.


ПЕРЕНОС НА ДРУГОЙ ПК
--------------------
  1. Скопировать всю папку.
  2. Поставить Node.js один раз: https://nodejs.org (LTS).
  3. Подключить heart-helper.lua в OBS (Часть 1) и добавить источник (Часть 2).
  Пути править НЕ нужно. Картинка вшита — отдельные PNG не нужны.


ФАЙЛЫ
  heart-obs.html     — виджет (картинка внутри)
  server.js          — помощник (живые данные + само-выход при закрытии OBS)
  heart-helper.lua   — скрипт для OBS (запускает помощника вместе с OBS)
  _run-helper.vbs    — служебный (скрытый запуск помощника); сам не запускать
